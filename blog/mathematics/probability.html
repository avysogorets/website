Introduction
Definition [\(\sigma\)-algebra]. A \(\sigma\)-algebra \(\mathcal{A}\) on a set \(\Omega\) is a collection of its subsets such that \(\Omega\in\mathcal{A}\) and \(\mathcal{A}\) is closed under complements and countable unions. We call \((\Omega, \mathcal{A})\) a measurable space, and elements of \(\mathcal{A}\) events. Note that \(\mathcal{A}\) is also closed under countable intersections by the DeMorgan's law.
Definition [probability measure]. A probability measure \(\mathbb{P}\) on \(\sigma\)-algebra \((\Omega,\mathcal{A})\) is a function \(\mathbb{P}\colon\mathcal{A}\rightarrow[0,1]\) satisfying \(\mathbb{P}(\Omega)=1\) and sigma-additivity, i.e., given any countable collection \(\{A_i\}_{i\in I}\) of disjoint subsets \(A_i\in\mathcal{A}\), we have \(\mathbb{P}\left(\bigcup_{i\in I}A_i\right)=\sum_{i\in I}\mathbb{P}(A_i)\). The triplet \((\Omega, \mathcal{A}, \mathbb{P})\) is called a probability space.
Note that, in general, infinite sums require extra care as they may have different values for different rearrangements of the summands (see conditionally convergent series). However, the positivity assumption on \(\mathbb{P}\) makes the above infinite series absolutely convergent, and so its value is independent of the order of individual terms. The prefix sigma- in sigma-additivity highlights that the corresponding property holds for countably infinite collections \(\{A_i\}_{i\in I}\). In constrast, additivity ensures the condition only for finite collections of subsets.
Proposition [continuity of probabilities]. Given a monotone increasing sequence \(\{A_i\}_{i=1}^{\infty}\) of events (\(A_i\subseteq A_{i+1}\) for all \(i\)) with a limit \(A=\bigcup_{i=1}^{\infty}\) (often written as \(A_i\uparrow A\)), we have \(\lim_{n\rightarrow\infty}\mathbb{P}(A_n)=\mathbb{P}(A)\).
Proof.
    Define a sequence of events \(\{B_i\}_{i=1}^{\infty}\) where \(B_1=A_1\) and \(B_i = A_{i}\setminus A_{i-1}\) for \(i>1\) (note that we implicitly assume that \(B_i\in\mathcal{A}\), which can be shown using the definition of \(\mathcal{A}\)). Since \(B_i\cap B_j = \emptyset\) whenever \(i\ne j\), sigma-additivity of the probability yields \(\mathbb{P}(A_n) = \mathbb{P}\left(\bigcup_{i=1}^{n} B_i\right)\)\(=\sum_{i=1}^{n}\mathbb{P}(B_i)\). Taking the limit on both sides, we recover \(\lim_{n\rightarrow\infty}\mathbb{P}(A_n)=\sum_{n=1}^{\infty}\mathbb{P}(B_n)\), which is \(\mathbb{P}\left(\bigcup_{i=1}^{\infty}B_i\right)=\mathbb{P}(A)\) by the application of sigma-additivity once again.
Definition [independence]. Two events \(A, B\in\mathcal{A}\) are independent (with respect to the given probability measure \(\mathbb{P}\)) if \(\mathbb{P}(A\cap B)=\mathbb{P}(A)\mathbb{P}(B).\) A collection of events \(\{A_i\}_{i\in I}\) is independent if for any finite subset \(J\subseteq I\), we have \(\mathbb{P}\big(\bigcap_{j\in J}A_j\big)=\prod_{j\in J}\mathbb{P}(A_j)\).
We can make a few basic observations using this definition. First, an event that holds with probability \(1\) is independent to any other event. Furthermore, if two disjoint events both have non-zero probability, they cannot be independent.
Definition [conditional probability]. Given an event \(A\in\mathcal{A}\) of a probability space \((\Omega,\mathcal{A},\mathbb{P})\) with \(\mathbb{P}(A)>0\), we define a conditional probability measure \(\mathbb{P}(\cdot|A)\colon\mathcal{A}\rightarrow[0,1]\) as \(\mathbb{P}(B|A)=\mathbb{P}(A\cap B)/\mathbb{P}(A)\).
Proof.
    We need to show that conditional probability function is indeed a well-defined probability measure. Clearly, \(\mathbb{P}(\Omega|A) = 1\). Given a countable collection of disjoint events \(\{B_i\}_{i\in I}\), we have \[\mathbb{P}\left(\bigcup_{i\in I}B_i\Big|A\right) = \frac{\mathbb{P}\left(\bigcup_{i\in I}B_i\cap A\right)}{\mathbb{P}(A)}=\frac{\sum_{i\in I}\mathbb{P}(B_i\cap A)}{\mathbb{P}(A)}=\sum_{i\in I}\mathbb{P}(B_i|A),\] as desired.
Note that \(\mathbb{P}(B|A) = \mathbb{P}(B)\) is \(A,B\) are two independent events.
Proposition [chain rule]. Let \(\{A_i\}_{i=1}^{n}\) be a collection of events such that \(\mathbb{P}\left(\bigcap_{i=1}^{n-1}A_i\right)>0\). Then, \[\mathbb{P}\left(\bigcap_{i=1}^nA_i\right)=\mathbb{P}(A_1)\mathbb{P}(A_2|A_1)\mathbb{P}(A_3|A_1\cap A_2)\ldots\mathbb{P}\left(A_n\Big|\bigcap_{i=1}^{n-1}A_i\right).\]
Proof.
    Using the definition of conditional probability.
Proposition [total probability]. Given an event \(A\in\mathcal{A}\) and a partition \(\Omega=\bigcup_{i\in I}E_i\) by countably many disjoint events \(E_i\), we have \(\mathbb{P}(A)=\sum_{i\in I}\mathbb{P}(A|E_i)\mathbb{P}(E_i)\).
Proof.
    By definition of conditional probability and sigma-additivity of probability measures.
Proposition [Bayes theorem]. Given an event \(A\) and a partition \(\Omega=\bigcup_{i\in I}E_i\) by countably many disjoint events with \(\mathbb{P}(E_i)>0\), we have \[\mathbb{P}(E_i|A)=\frac{\mathbb{P}(A|E_i)\mathbb{P}(E_i)}{\sum_{j\in I}\mathbb{P}(A|E_j)\mathbb{P}(E_j)}.\] Proof.
    By definition of conditional probability and the law of total probability.
As the name suggests, Bayes theorem is the basis of Bayesian inference. In this context, the Bayes rule is commonly known as \(p(H|E)\propto p(E|H)p(H)\) where \(H\) is the hypothesis (e.g., model parameters) and \(E\) is the evidence (e.g., data).

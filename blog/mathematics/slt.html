Introduction
Statistical learning theory is intended to give a basic statistical framework for the learning problems that typically arise in data science. Let's start by formally introducing notation of the standard learning setup. Let \((X,Y)\sim P\) be a random input-target pair generated by the data distribution \(P\) with \(\text{range}(X)=\mathcal{X}\) and \(\text{range}(Y)=\mathcal{Y}\) are the spaces of all possible values of our input and target variables, respectively. For example, for the MNIST image classification task, we can put \(\mathcal{X}=[255]^{28\times28}\) to be the space of all \(28\times 28\) black-and-white images, and \(\mathcal{Y}=\{0\ldots 9\}\) to be a collection of all possible class identities (labels 0 through 9). Let \(f\colon \mathcal{X}\rightarrow\mathcal{A}\) be the decision function (i.e., the model) that maps inputs to outputs in the action space \(\mathcal{A}\) (in most practical scenarios \(\mathcal{A}=\mathcal{Y}\)).
